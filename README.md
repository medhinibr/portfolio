# Portfolio

A premium, modern developer portfolio showcasing system architectures, software engineering capabilities, and cloud engineering practices.

## Core Stack
- **Framework**: React.js with Vite
- **Styling**: Tailwind CSS / Vanilla CSS
- **Animation**: Framer Motion for staggered reveals, letter compiling morphs, and transitions
- **Scroll Handling**: Lenis Smooth Scroll for kinetic desktop page scrolling
- **Development**: Node.js & npm tooling

## Key Features
- **Staggered Entrance Sequences**: Framer Motion entrance reveals staggered across text components, action buttons, and background glows.
- **Dynamic Compilation Effect**: Interactive name letters displaying underlying technologist concepts on cursor hover with instant transition collapse on unhover.
- **Cinematic Backdrop**: Fixed fractal noise overlay with deep background glow gradients and large colorful mesh backlights.
- **Interactive Cursor Dot**: Mix-blend-difference spring physics custom cursor that expands when hovering interactive elements.
- **Full Viewport Header Bounds**: Floating theme toggle and drawer icon pushed to the extreme viewport borders for modern aesthetic balancing.
